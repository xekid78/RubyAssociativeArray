# RubyHash
ハッシュ（連想配列）

## 処理
中ボスを出力する

## コード
```
enemy_array = {"ザコ" => "スライム","中ボス" => "ドラゴン","ラスボス" => "魔王"}
puts enemy_array["中ボス"]
```

## 出力結果  
```
ドラゴン
```
